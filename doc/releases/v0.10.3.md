# oradba v0.10.3

Oracle Database Administration Toolset

## Installation

```bash
curl -o oradba_install.sh -L https://github.com/oehrlis/oradba/releases/download/v0.10.3/oradba_install.sh
chmod +x oradba_install.sh
sudo ./oradba_install.sh
```

## Help System Usage

After installation, get help with:

```bash
# Quick reference
alih                          # Alias help
version                       # Version info
oraenv.sh --help             # Environment setup help

# Comprehensive help
oradba help                   # Main help menu
oradba help aliases           # Detailed alias reference
oradba help scripts           # Available scripts
oradba help variables         # Environment variables
oradba help config            # Configuration system

# Online documentation
oradba help online            # Opens https://oehrlis.github.io/oradba/
```

## Requirements

- Linux or macOS with Bash 4.0+
- Oracle Database 11g or higher
- Valid `/etc/oratab` file

## Quick Start

```bash
# Install OraDBA
./oradba_install.sh

# Set environment for a database
source /opt/oradba/bin/oraenv.sh FREE

# View status
oraup

# Get help
oradba help

# Connect to database
sq
```

For detailed instructions, see the [Quick Start Guide](https://oehrlis.github.io/oradba/03-quickstart/).
