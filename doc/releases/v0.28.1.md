# Release v0.28.1 - CI scope cleanup (CF-001 follow-up)

**Release date:** 2026-06-26
**Previous version:** 0.28.0

## Summary

Hotfix: `oradba_install.sh` and `oraenv.sh` were in the CI arithmetic-lint scope since M2
but contained zero-start `(( var++ ))` patterns that were not cleaned at the time. This
release fixes all 7 remaining occurrences and applies `shfmt` to both files.

## Changes

### Fixed

- `oradba_install.sh`: replaced `((warnings++)) || true` (3 occurrences) and
  `((match_count++)) || true` (2 occurrences) with `var=$(( var + 1 ))`; applied shfmt
- `oraenv.sh`: replaced 2 `((counter++))` occurrences; applied shfmt

## Modified Files

- `src/bin/oradba_install.sh`
- `src/bin/oraenv.sh`
- `CHANGELOG.md`
- `VERSION`
