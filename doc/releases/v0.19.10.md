# Release Notes v0.19.10

**Release Date:** 2026-01-24  
**Type:** Patch Release (Bug Fixes)

## Overview

Version 0.19.10 delivers a focused bugfix update for DataSafe status detection
and improves plugin execution efficiency. This release also refines associated
tests to prevent false failures.

## What's Fixed

### ✅ DataSafe Status Detection Improvements

- `oradba_check_datasafe_status()` now accepts an optional instance name
- `oradba_get_product_status()` passes the instance name to DataSafe checks
- Plugin status checks receive instance context for more accurate detection

### ✅ Test Stability Improvements

- Updated `check_datasafe_status` tests to use direct `declare -F` validation
- Prevents false failures when functions are already sourced

## What's Changed

### ⚡ Plugin Execution Optimization

- Replaced subshell `cat` with bash builtin file read in `oradba_apply_oracle_plugin()`
- Reduces subshell creation during plugin execution

## Compatibility

- Fully backward compatible with OraDBA v0.19.x
- No configuration changes required

## Upgrade Notes

```bash
# Standard upgrade
bash oradba_install.sh --update

# Verify version
oradba_version.sh --check
```
