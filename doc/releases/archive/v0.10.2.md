# oradba v0.10.2

Oracle Database Administration Toolset

## Installation

```bash
curl -o oradba_install.sh -L https://github.com/oehrlis/oradba/releases/download/v0.10.2/oradba_install.sh
chmod +x oradba_install.sh
sudo ./oradba_install.sh
```

### Enhanced Validation Script

The `oradba_validate.sh` script now provides cleaner output and more comprehensive checks:

- **Regular mode**: Concise output showing only section headers and summary
- **Verbose mode**: Detailed test results for troubleshooting
- **New checks**: Validates `.install_info` and `.oradba.checksum` metadata files
- **Modification detection**: Shows count of files modified since installation
- **Better summaries**: Includes modification status in validation results

### CI/CD Improvements (from v0.10.1)

- **Smart Test Selection**: Only runs relevant tests based on changed files

- **Release Workflow**: Linting runs before tests for faster failure detection

## Requirements

- Linux or macOS with Bash 4.0+
- Oracle Database 11g or higher
- Valid `/etc/oratab` file

## Quick Start

```bash
# Install OraDBA
./oradba_install.sh

# Set environment for a database
source /opt/oradba/bin/oraenv.sh FREE

# View status
oraup

# Connect to database
sq
```

For detailed instructions, see the [Quick Start Guide](https://oehrlis.github.io/oradba/03-quickstart/).
