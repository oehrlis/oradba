## Phase 3: Testing Review and Optimization

**Location:** `tests/` directory  
**Goal:** Efficient, focused test suite (reduce from 900+ tests if needed)

### Task 3.1: Test Inventory and Analysis (2-3 hours)

**Action:** Analyze current test suite

```bash
cd tests/
find . -name "*.bats" -type f -exec wc -l {} + | sort -n
bats --count tests/*.bats
```

**Review Questions:**

- What's actually being tested?
- Are there duplicate tests?
- Are there obsolete tests (old features)?
- Are critical paths covered?
- Are tests too granular or redundant?

### Task 3.2: Identify Redundant Tests (2-3 hours)

**Action:** Find tests that can be consolidated or removed

**Redundancy Patterns:**

- Multiple tests for same function with trivial variations
- Tests for deprecated/removed features
- Tests that duplicate integration test coverage
- Overlapping unit tests

**Create List:**

```markdown
## Redundant Tests to Remove/Consolidate

### test_common.bats
- Lines 100-150: Duplicate logging tests (keep 3, remove 5)
- Lines 200-250: Old oratab parsing (now obsolete)

### test_config.bats
- Lines 50-100: Redundant with environment tests
```

### Task 3.3: Update testmap.yml (1-2 hours)

**Action:** Update test mapping for current architecture

**Structure:**

```yaml
test_suites:
  unit_tests:
    - test_common.bats: "Core utility functions"
    - test_parser.bats: "Environment parser library"
    - test_builder.bats: "Environment builder library"
    # ... etc
  
  integration_tests:
    - test_environment_integration.bats
    - test_oracle_homes_integration.bats
  
  coverage:
    target: 80%
    critical_paths: 100%
```

### Task 3.4: Clean Up Test Suite (3-4 hours)

**Action:** Remove/consolidate redundant tests

**Process:**

1. Create backup branch: `git checkout -b test-cleanup-backup`
2. Remove redundant tests
3. Consolidate related tests
4. Verify all critical paths still covered
5. Run full test suite: `make test`
6. Document coverage gaps

**Target:** Reduce to ~500-600 focused, meaningful tests

### Task 3.5: Run Full Test Suite (1 hour)

**Action:** Comprehensive testing before release

```bash
make test
make test-integration  # if available
make lint
make shellcheck
```

**Document results and fix any failures**
