# OraDBA Build Artifacts

Build output directory for generated files and distribution packages.

## Contents

This directory contains:

- Built installers (e.g., `oradba_install.sh`)
- Release tarballs (e.g., `oradba-x.y.z.tar.gz`)
- Temporary build artifacts

## Usage

Generated by `make build`:

```bash
make build
# Creates: build/oradba-x.y.z.tar.gz
# Creates: dist/oradba_install.sh
```

## Cleanup

Clean build artifacts:

```bash
make clean
```
